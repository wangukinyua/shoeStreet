 git init - initializes a git repo
 git add{filename} - tracks the file added for changes
 git add. - adds all files to be tracked
 git status - checks the state of your working tree

 U Flag stands for an untracked file
 A Flag stands for a file that is being tracked by git
 M Flag (GREEN) stands for a modified file without errors
 M Flag (RED) stands for modified file with errors
